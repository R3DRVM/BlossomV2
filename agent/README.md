# Blossom Agent

Backend agent service for Blossom AI Trading Copilot.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev:agent

# Build
npm run build

# Start production server
npm start
```

## API Endpoints

- `POST /api/chat` - Chat with Blossom agent
- `POST /api/strategy/close` - Close a strategy
